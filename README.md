# BreakLine Website

This is a static marketing site for BreakLine and the BreakCore trading engine.

## Publish Free

You can host this site for free on any of these:

- GitHub Pages
- Netlify
- Vercel

## Files

- `index.html`
- `styles.css`

No build step is required.

## Local Preview

From the `website` folder:

```bash
python3 -m http.server 8080
```

Then open:

```text
http://localhost:8080
```

## Before Publishing

Update the placeholder contact email in `index.html`:

- `you@example.com`

If you want to pitch this as an acquisition asset, you can also replace the contact button with your Acquire.com listing URL.
